PROJECT NAME: Document Object Model SBA

DESCRIPTION:
Lucky Number Grid Game is an interactive browser-based game built with HTML, CSS, and JavaScript. Players flip mystery boxes on a grid to reveal hidden random numbers. The first number revealed becomes the player’s lucky number, and the objective is to continue flipping boxes until another box matches that lucky number.

The project focuses heavily on DOM manipulation, event listeners, animations, audio integration, and game logic using JavaScript.

FEATURES:

- Dynamic 100-box interactive grid
- Random number generation
- Flip animations on box click
- Lucky number system
- Win detection logic
- Sound effects on clicks
- Background suspense music
- Navigation bar interactions
- Form validation system
- Error handling and display system
- Responsive flexbox layout

TECHNOLOGIES USED:

- JavaScript
- HTML
- CSS
- Document Object Model
- Browser Object Model
- VS Code
